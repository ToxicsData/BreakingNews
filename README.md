## Breaking News
This project is licensed 

<h4>Clone to project</h4>
```commandline
git clone git@github.com:ToxicsData/BreakingNews.git
```

<h4>Virtualenv</h4>
```commandline
virtualenv venv
```

<h4>Activate</h4>
```commandline
source venv/bin/activate
```

<h4>Install</h4>
```commandline
pip install -r requirements.txt
```


<h4>Migrations</h4>
```commandline
./manage.py makemigrations && ./manage.py migrate

```


<h4>Migrations</h4>
```commandline
./manage.py runserver

```
